this repo will be u📥 1. Clone the repository
git clone https://github.com/UT01000100/knowledgesharingITD.git
📂 2. Move into the project folder
cd knowledgesharingITD
🔍 3. Check repository status
git status
🌿 4. Create a new branch (VERY IMPORTANT)

Each user works in their own branch:

git checkout -b user1-branch

(Example: user2, user3, etc.)

✏️ 5. Make changes
Edit files
Add new files
Example: edit README or democode.py
➕ 6. Stage changes
git add .
💾 7. Commit changes
git commit -m "User1 updates README"
🚀 8. Push branch to GitHub
git push origin user1-branch
🔁 9. Create Pull Request (on GitHub)

On GitHub:

Go to repo
Click Compare & pull request
Submit PR
